# 3D Pipes Screensaver

A faithful recreation of the iconic Windows 95 3D Pipes screensaver, built as a single HTML file with Three.js.

**[Live Demo](https://afraazali42.github.io/3d-pipes-screensaver/)**

## Usage

Open `index.html` in any browser. No build step, no dependencies to install.

### Embed in your site

```html
<iframe src="https://afraazali42.github.io/3d-pipes-screensaver/" style="width:100%;height:100vh;border:none;"></iframe>
```

Or copy the `index.html` into your own project and serve it directly.
